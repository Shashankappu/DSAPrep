## ✨ Features

- **🔍 Search & Filter**: Search by name, filter by category, difficulty, source (Blind 75, Neetcode 150, Striver SDE, Google Fav)
- **📋 Each Problem Card Contains**:
  - Problem name & number
  - Input/Output examples
  - Constraints
  - Simple explanation
  - Step-by-step approach
  - **C++ solution with syntax highlighting**
  - Time & Space complexity
  - **YouTube video solution link**
- **✅ Progress Tracking**: Mark problems as solved, bookmark favorites (persisted in localStorage)
- **📊 Dashboard**: Progress bars by difficulty, overall completion ring
- **⏰ Countdown Timer**: Days until your interview
- **🌙 Dark/Light Theme**
- **📱 Responsive**: Works great on mobile (Android browser)
- **⌨️ Keyboard Shortcuts**: `Ctrl+K` for search, arrow keys to navigate problems

## 🏃 How to Run

The app is currently running at: **http://localhost:8080**

To restart later:
```bash
cd path/to/DSAPrep
python -m http.server 8080
```

Then open `http://localhost:8080` in your browser.

> [!TIP]
> On Android, open the URL in Chrome and tap "Add to Home Screen" to install it as a PWA app!
